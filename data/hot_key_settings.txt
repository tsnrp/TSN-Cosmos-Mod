hot_key_settings.txt

This file contains the hot key mapping for Artemis Cosmos.
You may edit this file, to change the hot keys to your liking.
However, this file can get re-generated by the program, though it should not overwrite your changes.
Look at the bottom of this file for a list of valid names for the different keys on the keyboard.

The first key tag is always the primary key.  Subsequent tags are 'modifiers', like shift or control, which have to
be held down for the primary key press to register.  You can use a 'NOT' tag before a modifier, to invert its
function (NOT KEY_CONTROL means the primary key won't register, if the control key is held down).  Up to five
modifiers may be used on a key.  Key tags are separated by spaces.
